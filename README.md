# AI Resume Analyzer

An AI-powered web application that evaluates resumes against job descriptions, calculates a match score, identifies missing skills, and provides personalized recommendations.

---

## Features

✅ PDF Resume Upload

✅ Resume Text Extraction

✅ Skill Detection

✅ Job Description Matching

✅ Match Score Calculation

✅ Missing Skill Identification

✅ AI Recommendations

---

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Bootstrap
- PyMuPDF
- Git/GitHub

---

## Project Workflow

1. Upload Resume (PDF)
2. Paste Job Description
3. Analyze Resume
4. View:
   - Match Score
   - Skills Found
   - Missing Skills
   - AI Recommendations

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/resume-analyzer.git
```

Move into project directory:

```bash
cd resume-analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
